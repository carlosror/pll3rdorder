Upload noise file:
  • Enter PLL parameters and submit by clicking on "Evaluate PLL".
  • To upload noise sources data, first download the default noise sources Excel file, edit it, then upload it. If no noise file is uploaded, default noise data is used.
  • Following the phase noise plot, enter an integration interval to estimate the RMS phase error.
  • Checking "Generate Excel report" and submitting prints results to an Excel file.
  • NO DATA IS STORED.
